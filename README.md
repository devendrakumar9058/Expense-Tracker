# 💰 Expense Tracker

A Python-based **Expense Tracker** application that helps users manage daily income and expenses efficiently. It allows tracking, categorizing, and analyzing spending to improve financial management.

---

## 🚀 Features

* Add, view, and delete expenses
* Categorize transactions (Food, Travel, Bills, etc.)
* Track total spending
* Simple and user-friendly structure
* Scalable modules (OCR, payments, AI integration)

---

## 🛠️ Tech Stack

* Python
* Pandas
* (Optional) Streamlit / Flask
* API Integration (Gemini / OCR)

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
```

2. Create virtual environment:

```
python -m venv .venv
source .venv/bin/activate   # (Linux/Mac)
.venv\Scripts\activate      # (Windows)
```

3. Install dependencies:

```
pip install -r requirements
```

4. Setup environment variables:

```
cp .env.example .env
```

(Add your API keys inside `.env`)

---

## ▶️ Run Project

```
python run_project.py
```
